## BAGIAN B — PERBAIKAN TEKNIS & LAYOUT

### B1. Facilities.tsx — ganti icon emoji jadi Lucide icon
Emoji rendering tidak konsisten di berbagai OS/browser (device Android, iOS, Windows semua render beda) — untuk situs production yang presisi, ganti ke icon set yang konsisten.

Tambahkan import di baris paling atas:
```tsx
import { ShieldCheck, Plane, Building2, Bus, BookOpen, Briefcase, Armchair, Users } from 'lucide-react';
```

Ganti render icon (baris ~79-83), dari:
```tsx
<div className="w-[38px] h-[38px] rounded-[10px] bg-white flex items-center justify-center mb-[16px] text-[18px]">
  {f.icon}
</div>
```
Jadi:
```tsx
<div className="w-[38px] h-[38px] rounded-[10px] bg-white flex items-center justify-center mb-[16px]">
  <f.icon size={20} color="#1B1B36" strokeWidth={2} />
</div>
```
(Array `facilities` sudah pakai referensi komponen icon, bukan string emoji — lihat Bagian A4 di atas.)

### B2. Gallery.tsx — siapkan slot gambar asli, bukan cuma placeholder pattern
Ganti div placeholder (baris ~38-51) supaya bisa menampilkan foto asli begitu di-upload, dengan fallback graceful kalau file belum ada:
```tsx
<div
  className="rounded-xl overflow-hidden"
  style={{ aspectRatio: '1/1', background: '#F4F4F7' }}
>
  <img
    src={`/images/perlengkapan/${item.slug}.jpg`}
    alt={item.name}
    className="w-full h-full object-cover"
    onError={(e) => {
      e.currentTarget.style.display = 'none';
      e.currentTarget.parentElement!.style.background =
        'repeating-linear-gradient(135deg,rgba(27,27,54,0.045),rgba(27,27,54,0.045) 10px,rgba(27,27,54,0.08) 10px,rgba(27,27,54,0.08) 20px)';
    }}
  />
</div>
```
Tambahkan field `slug` di array `items` (baris 3-10), contoh:
```tsx
const items = [
  { name: 'Koper & Tas Umroh', slug: 'koper-tas' },
  { name: 'Mukena & Ihram', slug: 'mukena-ihram' },
  { name: 'Sajadah Travel', slug: 'sajadah-travel' },
  { name: 'Buku Doa & Manasik', slug: 'buku-doa-manasik' },
  { name: 'Seragam Kloter', slug: 'seragam-kloter' },
  { name: 'Kit Kesehatan', slug: 'kit-kesehatan' },
];
```
Buat folder `public/images/perlengkapan/` dengan file `README.txt` yang list 6 nama file di atas beserta rasio 1:1 yang dibutuhkan, supaya saya tinggal upload foto asli ke situ.

## BAGIAN C — OPSIONAL (boleh dilewati kalau waktu terbatas)

Saat ini semua warna (`#1B1B36`, `#292951`, `#6B6B85`, dll) di-hardcode langsung di setiap komponen sebagai inline style. Ini tidak masalah secara visual, tapi kalau suatu saat lo mau ganti warna brand lagi, harus cari-ganti manual di 13 file berbeda. Kalau mau lebih maintainable jangka panjang, bisa buat `src/lib/theme.ts`:
```tsx
export const COLORS = {
  navy: '#1B1B36',
  navyLight: '#292951',
  gray: '#6B6B85',
  white: '#FFFFFF',
  urgent: '#B5442E',
};
```
Lalu import dan pakai `COLORS.navy` dkk di semua komponen. Ini murni refactor teknis, tidak mengubah tampilan sama sekali — skip dulu kalau prioritas lo saat ini publish cepat, bisa dikerjakan belakangan.
