## KONTEKS

Buat halaman baru di route `/paket-umroh` yang menampilkan SEMUA kombinasi paket (tier × durasi × tanggal keberangkatan) dengan filter, berbeda dari section "Paket Umroh" di homepage yang cuma teaser 4 card. Reuse gaya card yang sudah ada (poster 4:5 + info + tags + harga + tombol Booking).

## 1. Tambahkan route baru — `src/App.tsx`

Tambahkan import dan route:
```tsx
import PaketUmroh from '@/pages/PaketUmroh';

// di dalam <Switch>, tambahkan SEBELUM <Route component={NotFound} />:

```

export default function PaketUmroh() {

  const [durFilter, setDurFilter] = useState<'Semua' | '9 Hari' | '12 Hari'>('Semua');
