## KONTEKS PROJECT

Buatkan landing page marketing untuk **Dreammecca Tour & Travel**, agen perjalanan Umroh berbasis di Jakarta Selatan. Target audiens: calon jamaah Umroh (25-55 tahun) yang mencari paket Umroh terpercaya, mayoritas traffic dari iklan Facebook/Instagram dan pencarian Google. Landing page harus terasa **premium, terpercaya, dan islami** — bukan seperti landing page startup tech.

## STACK TEKNIS

- **Next.js 14** (App Router, TypeScript)
- **Tailwind CSS** untuk styling
- **Framer Motion** untuk animasi (subtle, jangan berlebihan)
- **shadcn/ui** untuk komponen dasar (accordion, card, button, dialog)
- **next/image** untuk semua gambar (wajib, jangan pakai `<img>` biasa)
- **Lucide React** untuk icon set
- Deploy target: Vercel

Install dependencies di awal:
```bash
npx create-next-app@latest dreammecca --typescript --tailwind --app --no-src-dir
cd dreammecca
npx shadcn@latest init
npm install framer-motion lucide-react
```

## DESIGN SYSTEM

**Palet warna** (evoke premium + islami, bukan tech/startup):
- Navy dasar: `#0B1330` (background gelap, hero, footer)
- Navy sekunder: `#141B3D` (card di atas background gelap)
- Gold accent: `#C9A455` (CTA, highlight, border aktif)
- Gold terang (hover): `#E0C077`
- Cream/off-white: `#F8F6F0` (background section terang)
- Teks gelap: `#1A1A1A`
- Teks di atas navy: `#F5F3EC`

**Tipografi:**
- Heading: `Fraunces` (Google Fonts) — serif elegan, kesan premium travel, weight 500-600
- Body: `Inter` — weight 400-500
- Jangan pakai font tech/geometric seperti Space Grotesk di sini — beda brand identity dari project SaaS lo yang lain

**Prinsip visual:**
- Border radius konsisten: 12px untuk card, 8px untuk button
- Shadow subtle, jangan neon/glow
- Animasi masuk on-scroll pakai Framer Motion `whileInView`, durasi 0.4-0.6s, jangan bouncy
- Spacing generous antar section (min 80px vertical padding di desktop, 48px di mobile)
- Semua CTA utama pakai warna gold solid dengan teks navy gelap (kontras tinggi, jangan ghost button untuk primary CTA)

## DETAIL TEKNIS TAMBAHAN

- **SEO**: setiap halaman punya proper `metadata` export (title, description, OG image) di Next.js App Router. Title: "Dreammecca Tour & Travel — Paket Umroh Terpercaya & Berkah"
- **Performance**: lazy load semua image di bawah fold pakai `next/image` dengan `loading="lazy"`, video hero pakai `preload="metadata"`
- **Accessibility**: semua tombol punya `aria-label`, kontras warna gold-on-navy dicek AA compliant, form/accordion keyboard-navigable
- **Responsive breakpoints**: mobile-first, test di 375px, 768px, 1024px, 1440px
- **WhatsApp integration**: semua CTA WhatsApp pakai format `https://wa.me/62XXXXXXXXXX?text=` dengan pesan ter-encode sesuai context tombol (paket spesifik vs general inquiry)
- **Animasi**: gunakan Framer Motion `variants` yang reusable, satu file `lib/animations.ts` untuk fade-up, stagger children, dll — jangan inline animation object di setiap komponen
