## KONTEKS

Pendekatan pattern-di-dalam-kurva menyebabkan seam/potongan kasar di beberapa titik transisi (gambar pattern di kurva dan di section tidak align persis pixel-nya, terlihat jelas di transisi About-ke-Facilities). Daripada terus coba selaraskan posisi gambar yang rawan meleset, balik ke pendekatan SOLID FILL yang smooth dan simpel — dijamin tidak ada seam karena tidak ada gambar yang perlu nyambung.

## GANTI ISI `src/components/SectionDivider.tsx` TOTAL (versi final, solid fill)

```tsx
import React from 'react';

interface SectionDividerProps {
  fillColor: string; // hex warna solid, contoh: '#292951'
}

export default function SectionDivider({ fillColor }: SectionDividerProps) {
  return (
    <div className="w-full overflow-hidden leading-none" style={{ marginBottom: '-1px', marginTop: '-1px' }}>
      <svg viewBox="0 0 1440 80" className="w-full h-[44px] md:h-[64px]" preserveAspectRatio="none">
        <path d="M0,0 C400,80 1040,80 1440,0 L1440,80 L0,80 Z" fill={fillColor} />
      </svg>
    </div>
  );
}
```

Hapus semua referensi `overlayColor`, `maskImage`, dan `backgroundImage` dari komponen ini — sudah tidak dipakai lagi.
