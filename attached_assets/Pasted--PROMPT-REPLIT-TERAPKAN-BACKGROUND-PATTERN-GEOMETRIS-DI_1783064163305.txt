## PERUBAHAN PER FILE

Untuk setiap file di bawah, cari `style={{ background: '#1B1B36' }}` atau `style={{ background: '#292951' }}` pada elemen section utama, dan ganti jadi pola ini (sesuaikan warna overlay-nya: pakai rgba dari warna asli section tersebut supaya sub-brand section tetap konsisten):

### Hero.tsx (baris ~10)
```tsx

style={{ background: '#1B1B36' }}

/* SESUDAH */
```
