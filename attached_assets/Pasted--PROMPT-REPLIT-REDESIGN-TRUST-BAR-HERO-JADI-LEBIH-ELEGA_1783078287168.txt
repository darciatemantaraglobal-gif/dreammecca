## KONTEKS

Strip 3-kolom di bawah Hero sekarang asimetris — kolom 1 cuma teks, kolom 2 ada foto avatar + kata besar italic "PPIU", kolom 3 heading caps dua baris. Ganti jadi struktur konsisten: **icon bulat + headline + caption pendek** di ketiga kolom, pisah pakai card halus (bukan garis vertikal kaku), buang foto avatar.

## GANTI SELURUH BLOK TRUST BAR DI Hero.tsx

Cari blok 3-kolom (biasanya bagian paling bawah return statement, berisi "Direct Flight", avatar stack, dan "TAHUN MELAYANI"), ganti total jadi:

```tsx
import { Plane, ShieldCheck, Users } from 'lucide-react';
// tambahkan ke import lucide-react yang sudah ada di Hero.tsx

const trustItems = [
  {
    icon: Plane,
    title: 'Direct Flight, Maskapai Ternama',
    desc: 'Saudia Airlines, Garuda Indonesia & FlyDubai — tanpa transit berlebih.',
  },
  {
    icon: ShieldCheck,
    title: 'PPIU Resmi',
    desc: 'Berizin resmi Kementerian Agama RI, diawasi langsung.',
  },
  {
    icon: Users,
    title: 'Melayani Sejak 2025',
    desc: 'Dipercaya jamaah dari berbagai kota di Indonesia.',
    link: true,
  },
];

// ganti render-nya jadi:
<div className="grid grid-cols-1 md:grid-cols-3 gap-[16px] mt-[64px]">
  {trustItems.map((item, i) => (
    <div
      key={item.title}
      className="rounded-2xl p-[28px]"
      style={{
        background: 'rgba(255,255,255,0.035)',
        border: '1px solid rgba(255,255,255,0.08)',
      }}
    >
      <div
        className="w-[44px] h-[44px] rounded-full flex items-center justify-center mb-[18px]"
        style={{ background: 'rgba(255,255,255,0.08)' }}
      >
        <item.icon size={20} color="#fff" strokeWidth={1.75} />
      </div>
      <h3 className="text-[17px] font-bold leading-[1.3]" style={{ color: '#fff' }}>
        {item.title}
      </h3>
      <p className="text-[14px] leading-[1.55] mt-[8px]" style={{ color: 'rgba(255,255,255,0.6)' }}>
        {item.desc}
      </p>
      {item.link && (
        <a
          href="#paket"
          className="inline-flex items-center gap-[6px] text-[13px] font-bold uppercase tracking-[0.04em] mt-[16px] no-underline hover:opacity-75 transition-opacity"
          style={{ color: '#fff', borderBottom: '1px solid rgba(255,255,255,0.4)', paddingBottom: '2px' }}
        >
          Lihat Paket
        </a>
      )}
    </div>
  ))}
</div>
```

Catatan:
- Ketiga card sekarang punya struktur visual identik (icon bulat, headline bold, caption redup) — jauh lebih rapi dibanding sebelumnya yang tiap kolom formatnya beda sendiri-sendiri.
- Card kolom 3 (yang ada link "Lihat Paket") dikasih border-bottom tipis di link-nya sebagai affordance kalau itu bisa diklik, karena tanpa warna aksen (monokrom) butuh cara lain buat nunjukin elemen interaktif.
- Kalau project sudah pakai sistem animasi Framer Motion dari prompt sebelumnya, bungkus `<div className="grid ...">` ini dengan `motion.div` + `variants={staggerContainer}`, dan tiap card dalam `.map()` pakai `motion.div` + `variants={fadeUp}`, supaya konsisten dengan section lain.
