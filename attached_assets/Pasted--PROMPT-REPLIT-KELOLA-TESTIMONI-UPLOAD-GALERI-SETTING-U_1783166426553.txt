## BAGIAN B — Kelola Testimoni

### B1. Ganti `Testimonials.tsx` — fetch dari Supabase

```tsx
import { useQuery } from '@tanstack/react-query';
import { supabase } from '@/lib/supabase';

interface Testimonial {
  id: string;
  name: string;
  meta: string;
  quote: string;
  rating: number;
}

function useTestimonials() {
  return useQuery({
    queryKey: ['testimonials'],
    queryFn: async () => {
      const { data, error } = await supabase.from('testimonials').select('*').eq('is_active', true).order('sort_order');
      if (error) throw error;
      return data as Testimonial[];
    },
  });
}
```
Ganti array hardcoded `testimonials` dengan hasil `const { data: testimonials } = useTestimonials();` di komponen, dan `initials` untuk avatar bulat dihitung dari `name` (ambil huruf pertama tiap kata): `name.split(' ').map(w => w[0]).join('').slice(0,2).toUpperCase()`.

```tsx
async function uploadGalleryPhoto(file: File, caption: string) {
  const ext = file.name.split('.').pop();
  const path = `jamaah-${Date.now()}.${ext}`;
  const { error: uploadError } = await supabase.storage.from('gallery').upload(path, file);
  if (uploadError) throw uploadError;
  const { data } = supabase.storage.from('gallery').getPublicUrl(path);
  await supabase.from('gallery_photos').insert({ image_url: data.publicUrl, caption });
}
```

## BAGIAN D — Setting Umum (WA, Alamat, Sosial Media)

### D1. Buat `src/lib/useSiteSettings.ts`

```tsx
import { useQuery } from '@tanstack/react-query';
import { supabase } from '@/lib/supabase';

export function useSiteSettings() {
  return useQuery({
    queryKey: ['site-settings'],
    queryFn: async () => {
      const { data, error } = await supabase.from('site_settings').select('*').eq('id', 1).single();
      if (error) throw error;
      return data;
    },
    staleTime: 5 * 60 * 1000,
  });
}
```

### D2. Pakai hook ini di tempat yang sebelumnya hardcode nomor WA/alamat

Di `lib/whatsapp.ts`, `Footer.tsx`, `FinalCTA.tsx`, dan komponen lain yang punya nomor WA/alamat statis — ganti jadi ambil dari `useSiteSettings()`. Contoh pola:
```tsx
const { data: settings } = useSiteSettings();
const waNumber = settings?.whatsapp_number ?? '6281311506025'; // fallback kalau belum load
```
Untuk `lib/whatsapp.ts` yang berupa fungsi biasa (bukan komponen React, tidak bisa pakai hook langsung) — ubah `createWALink` supaya menerima `waNumber` sebagai parameter, dan komponen yang memanggilnya menyuplai nilai dari `useSiteSettings()`.
