FOLDER FOTO PERLENGKAPAN JAMAAH
================================

Upload foto perlengkapan ke folder ini dengan nama file berikut (format JPG, rasio 1:1 / square):

  koper-tas.jpg        → Foto koper + tas jinjing Dreammecca
  mukena-ihram.jpg     → Foto mukena (wanita) dan kain ihram (pria)
  sajadah-travel.jpg   → Foto sajadah lipat travel
  buku-doa-manasik.jpg → Foto buku doa & panduan manasik
  seragam-kloter.jpg   → Foto seragam/baju kloter jamaah
  kit-kesehatan.jpg    → Foto kit kesehatan (obat, dll)

Resolusi minimum yang disarankan: 800×800 px.
Kalau file belum di-upload, section Gallery otomatis tampil placeholder abu-abu — tidak ada broken image icon.
